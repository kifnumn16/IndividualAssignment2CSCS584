PROJECT DESCRIPTION

The Student Profile Management System is a dynamic, enhanced web based application for registration, Management and Storage of Student details. This system is built on the Model-View-Controller (MVC) architectural design pattern. The system connects a frontend user interface to a backend database container using standard Java Web technologies.

The application is built with a modern and responsive layout and displays the data in a clean manner by using a sleek and user friendly color scheme of coral and white instead of plain text tables. It provides seamless, continuous data persistence throughout each step of the profile lifecycles.

IMPLEMENTED FEATURES

1. Centralized Dashboard (Main Menu): Provides a minimalist, responsive entrance card featuring shortcuts to add a new profile or review existing lists.

2. Student Profile Registration: Captures text, multi-valued fields and expanded text block via standard HTML forms.

3. Dynamic Card-Layout Grid: Swaps traditional tabular layouts for clean, elevated individual profile cards displaying student records visually.

4. Profile Search: Features a live search filter box allowing instantaneous lookup of profiles matching input keys across Student IDs or Names.

5. Profile Filtering: Features an advanced filtering mechanism to instantly sort student directories according to specific academic Programmes .

6. Inline Record Modification: Offers full profile update forms pre-loaded with current row records while enforcing strict Read-only locks on structural primary keys .

7. Safe Confirmation Deletion: Integrates native JavaScript interactive validation prompts verifying removal confirmation before removing rows from database storage tables.
